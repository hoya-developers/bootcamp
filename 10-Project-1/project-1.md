## Hoya Devs Bootcamp Project 1

Congratulations on completing the first 5 weeks of Bootcamp! This project is designed to put together everything that you have learned so far.

### Project Description

For this project, you will design, code, and deploy a website built using HTML/CSS. Specifically, you will be building a portfolio website that you can use in the future to showcase the projects you work on, work experiences, and hobbies.

### Project Requirements

Your website should:

- Include a single HTML file and a single CSS file (index.html and index.css). For now, only worry about building a single page.
- Use Google Fonts to choose a font for your website.
- Include at least one image on your website.
- Use Flexbox for layouts.
- Use git to track your changes and make commits.
- Deploy your website using GitHub Pages.

### Project Help

Before you start your own development, we recommend you use Google to find some inspiration for your portfolio site.

After browsing some examples, here is the steps we recommend:

1. We strongly recommend you start by designing your website using Figma. It is much easier to write HTML/CSS code once you know what you want your website to look like.
2. Once you've designed your website, start by creating the necessary files in a directory. You'll have an index.html file and a index.css file, along with any images.
3. Begin by writing the HTML code. Once you have the structure of your website, you can start styling it using CSS.
4. Initialize a git repository in your project directory and make commits as you make progress.
5. Once you're done, create a github repository and push your code to it. Then, deploy your website using GitHub Pages.
